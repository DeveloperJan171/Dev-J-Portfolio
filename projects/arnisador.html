PROJECT OVERVIEW

Arnisador

Arnisador is a 3D educational martial arts game developed as our undergraduate thesis using Unity. The project combines Filipino martial arts education with interactive gameplay while demonstrating gameplay programming, animation systems, technical art, and collaborative game development through a complete playable experience.

Gameplay footage captured from the final thesis build showcasing combat, educational systems, UI, player progression, and core gameplay mechanics.

PROJECT OBJECTIVE

Teaching Filipino Martial Arts Through Gameplay

Rather than creating a traditional fighting game, our objective was to transform Filipino martial arts into an interactive educational experience. The project combines combat mechanics, learning materials, and player progression to encourage both engagement and cultural appreciation.

Educational content and gameplay progression were designed to complement one another throughout the player's experience.

GAME SYSTEMS

Core Gameplay Systems

The project contains multiple interconnected gameplay systems that support player progression—from menu navigation and tutorials to combat preparation, educational content, and stage progression.

Loading Screen

Introduces the player before entering the game world.

Main Menu

Combat Preparation

Interactive learning resource covering Filipino martial arts.

Level Selection

Progression system that unlocks gameplay environments.
